# Installing React Alt Esri Javascript Boilerplate


## Dependencies

### Node
`brew install node` (may require sudo)

### Gulp - Task Runner
`npm install -g gulp` (may require sudo)

### Bower - Front-End Package Manager
`npm install -g bower` (may require sudo)
